# 사용자 퀵 매뉴얼

## 1. 빠른 시작

1. 브라우저에서 `login.html` 접속
2. `dongkuk1 / 1234` 입력 또는 `테스트 계정 바로 입장` 클릭
3. 상단 `조회 연도` 선택 (2023~2030)
4. 필요한 탭으로 이동
5. 필요 시 `보고서 내보내기`
6. 사용 종료 시 `로그아웃`

## 3. 자주 쓰는 기능

### 연도 바꾸기
- 상단 `조회 연도`에서 선택
- 현재 탭의 모든 데이터가 즉시 갱신됨

### 표 정렬
- 표 열 제목 클릭 → 오름차순/내림차순 전환
- 정렬 활성 열에 ▲ 또는 ▼ 표시

### 보고서 저장
- 상단 `보고서 내보내기` 클릭
- DOCX 파일 자동 다운로드
- 포함: 실적, 구매, 배분, 등급 (커버 + 4개 섹션)

### 공지 확인
- `공지사항` 탭 이동
- 제목 클릭 → 상세 내용 모달
- `필독` 배지 공지 = 중요 공지

### 일정 확인
- `공지사항` 탭 → 우측 캘린더
- 날짜 클릭 → 해당일 일정 목록
- 유형별 색상 블록으로 시각 구분

### 일정 등록
- `+ 일정 등록` → 팀원/유형/날짜 선택 → 저장
- 휴가/교육/출장은 종료일 입력 가능

### 거래처 관리
- `거래처 관리` 탭 → `+ 거래처 등록`
- 코드 자동 생성, 성과율 자동 계산
- 신뢰등급 배지 색상: A(녹) A-(파) B+(주황) B(회)

## 4. 챗봇 사용법

### 열기/닫기
- 우측 하단 로봇 아이콘 클릭
- `✕` 버튼 또는 팝업 `✕`로 닫기

### 질문 예시

| 유형 | 질문 예시 |
| --- | --- |
| 월별 실적 | `7월 계획과 실적은?` |
| 분기 달성률 | `1분기 달성률` |
| 반기 구매 | `상반기 구매 현황` |
| 등급 비중 | `등급별 비중`, `국고하 비중` |
| 거래처 | `현대스크랩 현황`, `거래처별 실적` |
| 수입 | `수입 선적 현황`, `3월 수입 현황` |
| 공장 배분 | `인천 배분 현황`, `5월 배분` |
| 전체 요약 | `전체 현황 요약` |

### 기간 표현

| 입력 | 인식 |
| --- | --- |
| `7월` | 7월 |
| `3~6월` | 3~6월 범위 |
| `1분기` ~ `4분기` | 해당 3개월 |
| `상반기` / `하반기` | 1~6월 / 7~12월 |
| `전체` / `연간` / `올해` | 1~12월 |

### 팁
- 구체적 키워드 사용: 계획, 실적, 구매, 등급, 거래처, 수입, 배분
- 거래처명 직접 입력 가능: `현대스크랩 현황`
- 공장 지정 가능: `인천 배분`, `포항 실적`
- 추천 질문 칩 클릭으로 빠르게 질문
- `↺` 버튼으로 대화 초기화

## 5. 운영 담당자에게 요청할 항목

| 작업 | 설명 |
| --- | --- |
| 월별 계획/실적 데이터 갱신 | 엑셀 → 계획 그리드 붙여넣기 |
| 원본 실적 데이터 갱신 | 엑셀 A:E열 → 실적 그리드 붙여넣기 |
| 검수 등급 매핑 변경 | 상세등급 → 대분류 매핑 추가/삭제/복원 |
| 사용자 계정 관리 | 계정 생성, 비활성 처리, 비밀번호 초기화 |
| 공지 등록/수정/삭제 | 비밀번호 필요 |
| 데이터 번들 재생성 | Python 스크립트로 XLSX → JS 변환 |

## 6. 문제 해결 빠른 참조

| 증상 | 해결 방법 |
| --- | --- |
| 로그인 실패 | 아이디/비밀번호 확인, 비활성 계정 여부 확인 |
| 데이터가 안 보임 | 조회 연도 확인, 해당 연도 데이터 입력 여부 확인 |
| 차트 안 보임 | 새로고침(Ctrl+Shift+R), CDN(인터넷) 접근 확인 |
| 다운로드 실패 | 새로고침 후 재시도, 팝업 차단 확인 |
| 챗봇 이해 불가 | 구체적 키워드 + 기간 명시, 추천 질문 참고 |
| 다른 PC에서 데이터 없음 | 정상 동작 (브라우저별 로컬 저장, 동기화 없음) |
| 기본 공지 삭제 후 재등장 | 정상 동작 (시스템 기본 공지는 항상 표시) |
| 인쇄 시 헤더 안 보임 | 정상 동작 (인쇄 시 UI 요소 자동 숨김) |

## 7. 키보드/조작 팁

| 동작 | 효과 |
| --- | --- |
| 챗봇 입력 후 Enter | 메시지 전송 |
| 그리드 셀에서 Enter | 다음 셀로 이동 (포커스 해제) |
| 그리드 셀 클릭 | 기존 값 전체 선택 (덮어쓰기 용이) |
| 표 열 제목 클릭 | 정렬 전환 |
| 탭 버튼 클릭 | 해당 탭으로 이동 + URL 해시 갱신 |
| 로고 클릭 | 첫 탭(부재료실적 모니터링)으로 이동 |
| 비밀번호 눈 아이콘 | 비밀번호 표시/숨김 전환 |
