# 사용자 퀵 매뉴얼

## 1. 빠른 시작
1. `login.html` 접속
2. `dongkuk1 / 1234` 입력 또는 `테스트 계정 바로 입장`
3. 상단 `조회 연도` 선택
4. 필요한 탭으로 이동
5. 필요 시 `보고서 내보내기`
6. 사용 종료 시 `로그아웃`

## 2. 화면별 핵심 기능

| 메뉴 | 바로 확인할 내용 |
| --- | --- |
| 부재료실적 모니터링 | KPI, 월별 계획 대비 실적, 거래처 추이, 누계 상세 |
| 등급별현황/수입관리 | 구매량 추이, 등급 비교, 수입 선적 현황 |
| 거래처 관리 | 거래처 목록, 신뢰등급, 공급능력, 납품량 |
| 공지사항 | 공지 열람, 팀 일정 확인 |
| 사용자관리 | 사용자 목록, 활성/비활성 상태 |

## 3. 자주 쓰는 기능

### 연도 바꾸기
- 상단 `조회 연도`에서 선택
- 현재 탭 데이터가 즉시 갱신됨

### 표 정렬
- 표 제목 클릭
- 오름차순/내림차순 전환

### 보고서 저장
- 상단 `보고서 내보내기` 클릭
- DOCX 파일 생성

### 공지 확인
- `공지사항` 탭 이동
- 제목 클릭 후 상세 확인

### 일정 확인
- `공지사항` 탭 오른쪽 달력에서 날짜 클릭

### 챗봇 질문
- 우측 하단 챗봇 버튼 클릭
- 예시: `7월 계획과 실적은?`

## 4. 운영 담당자에게 요청할 항목
- 월별 계획/실적 데이터 갱신
- 원본 실적 데이터 붙여넣기
- 검수 등급 매핑 변경
- 사용자 계정 생성/비활성 처리
- 공지 등록/수정/삭제

## 5. 문제 해결
- 로그인 실패: 아이디/비밀번호 확인, 비활성 계정 여부 확인
- 데이터가 안 보임: 조회 연도 확인
- 다운로드 실패: 새로고침 후 다시 시도
