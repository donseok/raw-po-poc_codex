# 사용자 매뉴얼

## 1. 목적

이 문서는 동국제강 원료기획팀 실적 모니터링 대시보드의 사용법을 안내합니다. 화면 조회, 연도 전환, 차트/표 활용, 보고서 내보내기, 챗봇 질의, 공지/일정 관리 등 일상 사용 흐름을 기준으로 작성했습니다.

---

## 2. 접속과 로그인

### 2.1 접속 방법

1. 브라우저에서 `login.html`에 접속합니다.
2. 아이디와 비밀번호를 입력한 뒤 `로그인`을 누릅니다.
3. 기본 테스트 계정은 `dongkuk1 / 1234`입니다.

### 2.2 빠른 접속

- `테스트 계정 바로 입장` 버튼을 누르면 `dongkuk1` 계정으로 자동 로그인됩니다.
- 이미 로그인된 상태에서 `login.html`에 접속하면 대시보드 화면(`index.html`)으로 자동 이동합니다.

### 2.3 비밀번호 표시

- 비밀번호 입력란 오른쪽의 눈 아이콘(👁)을 클릭하면 입력 중인 비밀번호를 확인할 수 있습니다.
- 다시 클릭하면 마스킹(●●●●) 상태로 돌아갑니다.

### 2.4 로그인 오류

| 상황 | 메시지 |
| --- | --- |
| 아이디/비밀번호 불일치 | "아이디 또는 비밀번호가 올바르지 않습니다." |
| 비활성 계정 | "비활성화된 계정입니다. 관리자에게 문의하세요." |

오류 메시지는 아이디 또는 비밀번호 입력 시 자동으로 사라집니다.

### 2.5 로그아웃

- 헤더 우측의 `로그아웃` 버튼을 클릭합니다.
- 현재 세션이 종료되고 로그인 화면으로 이동합니다.

## 3. 메인 화면 구성

### 3.1 헤더 영역

| 항목 | 설명 |
| --- | --- |
| 현재 날짜 | 접속 시점의 날짜와 요일을 한국어 형식으로 표시합니다 (예: 2024년 3월 15일 금요일). |
| 조회 연도 | 드롭다운으로 연도를 선택합니다 (2023~2030). 변경 시 모든 차트, KPI, 표가 해당 연도로 갱신됩니다. |
| 사용자 표시 | `동국제강 {부서} \| {이름}{직급}님` 형식으로 표시됩니다 (예: 동국제강 원료기획팀 \| 이돈석팀장님). |
| 보고서 내보내기 | 현재 선택 연도 기준 DOCX 보고서를 생성합니다. |
| 로그아웃 | 현재 세션을 종료하고 로그인 화면으로 이동합니다. |

### 3.2 데이터 분석 기준 배너

차트 아래에 데이터 분석 기준 정보가 표시됩니다:
- 분석 기준 연도
- 조직명
- 원본 파일명
- 데이터 생성 시점
- 포함된 섹션 목록

### 3.3 탭 메뉴

탭 네비게이션은 화면 상단에 고정(sticky)되어 스크롤해도 항상 보입니다.

| 탭 | 주요 용도 |
| --- | --- |
| 부재료실적 모니터링 | 월별 계획/실적, 거래처 추이, 핵심 포인트 확인 |
| 등급별현황/수입관리 | 구매량, 등급 구성, 공장 배분, 수입 선적 현황 확인 |
| 거래처 관리 | 거래처 목록 조회 및 등록/수정/삭제 |
| 공지사항 | 공지 열람 및 등록, 팀 일정 확인 |
| 사용자관리 | 사용자 목록 조회 및 등록/수정/삭제 |

**URL 기반 탭 이동**: 브라우저 주소에 `?tab=plan` 또는 `#plan` 형식으로 직접 탭을 지정할 수 있습니다. 북마크에 저장하면 해당 탭으로 바로 이동합니다.

### 3.4 로고 클릭

좌측 상단 로고를 클릭하면 `부재료실적 모니터링` 탭(첫 화면)으로 이동합니다.

## 4. 공통 사용 방법

### 4.1 조회 연도 변경

1. 헤더의 `조회 연도` 드롭다운을 엽니다.
2. 원하는 연도를 선택합니다 (2023~2030 범위).
3. 현재 활성 탭의 차트, KPI, 표가 자동으로 갱신됩니다.
4. 원본 실적 데이터, 계획 데이터 등 연도별 저장 데이터도 해당 연도 기준으로 전환됩니다.
5. 모든 캐시가 무효화되고 새로 계산됩니다.

### 4.2 표 정렬

정렬 가능한 표는 열 제목을 클릭하면 오름차순/내림차순으로 전환됩니다.

- 같은 열을 다시 클릭하면 정렬 방향이 반전됩니다.
- 다른 열을 클릭하면 해당 열 기준 오름차순으로 초기화됩니다.
- 정렬 활성 열에는 `▲`(오름차순) 또는 `▼`(내림차순) 표시가 나타납니다.
- `no-sort` 표시가 있는 열은 정렬이 불가합니다.

정렬 시 다음 값들이 자동 인식됩니다:
- `N월` 형식 → 월 순서로 정렬
- 쉼표가 포함된 숫자 (예: `1,234,567`) → 숫자로 변환하여 정렬
- 퍼센트 값 (예: `85.3%`) → 숫자로 변환하여 정렬
- 그 외 → 한국어 가나다 순 텍스트 정렬

### 4.3 보고서 내보내기

1. 헤더의 `보고서 내보내기` 버튼을 클릭합니다.
2. 시스템이 모든 차트를 캡처한 뒤 DOCX 파일을 생성합니다.
3. 파일이 자동으로 다운로드됩니다.

**파일명**: `동국제강_원료기획팀_{연도}년_보고서_{날짜}.docx`

**포함 내용**:
- 커버 페이지
- 부재료실적 모니터링 (KPI, 차트 2개, 월별 표, 거래처 표)
- 구매실적 (KPI, 차트, 월별 표)
- 공장배분 (KPI, 인천/포항 차트, 배분 표)
- 등급별현황/수입관리 (KPI, 차트 2개)

**비포함**: 거래처 관리 상세, 공지사항, 팀 일정, 사용자 정보

**주의**: 보고서 생성 중에는 화면이 잠시 깜빡일 수 있습니다 (숨겨진 탭의 차트를 렌더링하기 위함).

### 4.4 숫자 표시 형식

| 형식 | 적용 | 예시 |
| --- | --- | --- |
| 일반 숫자 | 표, KPI | `1,234,567` (천 단위 쉼표) |
| 만 단위 | 축약 표시 | `123.4만` |
| 억 단위 | 축약 표시 | `12.3억` |
| 백분율 | 달성률, 비중 | `95.2%` |

## 5. 탭별 사용법

### 5.1 부재료실적 모니터링

이 화면에서는 월별 목표 대비 실적을 한 번에 확인할 수 있습니다.

#### KPI 카드 (4개)

| KPI | 색상 | 설명 |
| --- | --- | --- |
| 연간 목표 | 네이비 | 전체 월 계획 합계 |
| 누계 실적 | 주황 | 마지막 유효 월까지의 실적 누계 |
| 계획 대비 달성률 | 녹색 | 누계 실적 / 누계 계획 × 100 |
| 거래처 평균 성과율 | 노랑 | 거래처 관리 탭 등록 거래처의 평균 성과율 |

#### 차트

| 차트 | 유형 | 설명 |
| --- | --- | --- |
| 월별 계획 대비 실적 | 막대 그래프 | 월별 계획(밝은 색)과 실적(진한 색) 비교. 각 막대 위에 수치 표시 |
| 월별 주요 거래처 납품 추이 | 영역 차트 | 상위 3개 거래처의 월별 납품량 추이. 곡선 보간 |

차트의 데이터 포인트에 마우스를 올리면 상세 수치가 툴팁으로 표시됩니다.

#### 핵심 체크 포인트

변동이 큰 구간이나 주목할 지표를 자동 요약합니다:
- **상단 변동 월**: 실적이 가장 높은 월과 계획 대비 차이
- **주의 구간**: 달성률이 가장 낮은 월과 계획 대비 차이
- **계획 미달 월 수**: 실적이 계획에 미치지 못한 월 수

#### 월별 누계 실적 상세 표

정렬 가능한 표에 다음 열이 표시됩니다:
- 월, 계획, 실적, 차이, 누계계획, 누계실적, 달성률(%)

#### 접힌 입력 패널 (3종)

다음 3개의 접힌 패널은 주로 운영 담당자가 사용합니다. 제목을 클릭하면 펼쳐집니다.

1. **월별 계획 대비 실적 데이터**: 엑셀에서 계획/실적 데이터를 붙여넣는 4행 × 12열 그리드
2. **거래처 납품 실적관리 데이터 (인천공장)**: 원본 거래 데이터를 붙여넣는 5열 그리드 (페이징 지원)
3. **검수 등급 매핑 관리**: 상세 등급 → 대분류 매핑 설정

일반 사용자는 조회 위주로 사용하고, 데이터 갱신이 필요하면 운영 담당자에게 요청하는 것을 권장합니다.

### 5.2 등급별현황/수입관리

이 화면에서는 구매량, 등급 구성, 공장 배분, 수입 진행 현황을 확인합니다.

#### KPI 카드

| KPI | 설명 |
| --- | --- |
| 월별 구매량 총합 | 전체 등급 물량 합계 |
| 누계 입고금액 | 총 입고금액 |
| 구매량/입고금액 비율 | 물량 대비 금액 비율 |
| 국고하+선반설 비율 | 저회전 등급 비중 (전년 대비 증감 표시) |

#### 차트

| 차트 | 유형 | 설명 |
| --- | --- | --- |
| 월별 구매량 추이 | 막대+선 복합 | 구매량(막대, 보라) + 입고금액(선, 주황). Y축 2개 |
| 인천공장 등급 구성 | 도넛 | 인천 공장 등급별 비중 (중앙 54% 컷아웃) |

#### 구매실적 상세

| 항목 | 설명 |
| --- | --- |
| 누계 구매량 | 연간 총 구매량 |
| 누계 입고금액 | 연간 총 입고금액 |
| 평균 매입 단가 | 총 입고금액 / 총 구매량 |
| 월 평균 거래처 수 | 월별 유니크 거래처 수의 평균 |
| 최대/최소 구매량 월 | 구매량 기준 최대/최소 월 |
| 최고/최저 단가 월 | 평균 단가 기준 최고/최저 월 |

#### 공장 배분 현황

인천/포항 공장별:
- 연간 계획, 연간 실적, 달성률
- 등급 도넛 차트 (국고상/국고중/국고하/선반설/기타)
- 진행바로 달성률 시각화
- 국고하+선반설(저회전) 비율과 전년 대비 증감

#### 등급별 비교 표

현재 연도와 전년도의 등급별 물량, 비중, 비중 차이를 비교합니다.

#### 월별 구매실적 상세 표

월, 입고량, 입고금액, 평균 단가, 거래처 수를 정렬 가능한 표로 표시합니다.

#### 수입 계약/선적 현황

수입 거래의 상태별 목록을 표시합니다.

| 상태 | 배지 색상 |
| --- | --- |
| 도착 | 녹색 |
| 운송중 | 파랑 |
| 선적 | 주황 |
| 계약 | 회색 |

### 5.3 거래처 관리

거래처 관리 탭에서는 등록된 공급업체 정보를 확인하고 관리할 수 있습니다.

#### KPI 영역

| KPI | 설명 |
| --- | --- |
| 전체 거래처 | 등록된 거래처 수 |
| 월 공급능력 합계 | 전체 거래처의 월 공급능력 합산 |
| 연간 납품량 합계 | 전체 거래처의 금년 납품량 합산 |
| 평균 성과율 | 전체 거래처의 성과율 평균 |

#### 거래처 목록 표

| 열 | 설명 |
| --- | --- |
| 코드 | 거래처 고유 코드 (예: S001) |
| 거래처명 | 공급업체 이름 |
| 지역 | 소재 지역 |
| 신뢰등급 | A, A-, B+, B (배지 색상으로 구분) |
| 월 공급능력 | 월간 최대 공급 가능 톤수 |
| 금년 납품량 | 올해 실제 납품 톤수 |
| 성과율 | 납품량 / (월 공급능력 × 12). 색상 막대로 표시 |
| 관리 | 수정/삭제 버튼 |

#### 거래처 등록/수정

- `+ 거래처 등록` 버튼으로 모달이 열립니다.
- 코드는 신규 등록 시 자동 제안됩니다 (기존 최대 번호 + 1).
- 수정 모드에서는 코드가 읽기 전용입니다.

### 5.4 공지사항

공지사항 탭에서는 공지를 조회하고 등록할 수 있습니다.

#### 공지 목록

- 제목을 클릭하면 상세 내용 모달이 열립니다.
- `필독` 공지는 상단에 고정되고 빨간 `필독` 배지가 표시됩니다.
- 일반 공지는 생성일 역순으로 표시됩니다.
- 페이지당 10건 표시됩니다.

#### 공지 상세 보기

모달에서 다음 정보를 확인합니다:
- 제목
- 작성자, 작성일
- 본문 내용
- 수정/삭제 버튼 (비밀번호 필요)

#### 공지 등록

1. `+ 등록` 클릭
2. 제목, 내용, 작성자, 비밀번호 입력
3. 필요 시 `필독` 체크
4. `저장`

**비밀번호 주의**: 등록 시 입력한 비밀번호는 이후 수정/삭제 시 필요합니다. 비밀번호를 분실하면 해당 공지를 수정/삭제할 수 없습니다.

### 5.5 팀 일정관리

공지사항 탭 오른쪽에 팀 일정 캘린더가 있습니다.

#### 캘린더 보기

- 월 단위 6주 그리드로 표시됩니다.
- 이전/다음 월 이동 화살표가 있습니다.
- 오늘 날짜는 네이비 배경으로 강조됩니다.
- 일요일은 빨간색, 토요일은 파란색으로 표시됩니다.
- 일정이 있는 날은 유형별 색상 블록이 표시됩니다 (최대 3개, 초과 시 `+N`).

#### 일정 등록

1. `+ 일정 등록` 클릭
2. 팀원 선택 (이돈석, 박영수, 박지영, 유남, 김민재)
3. 일정 유형 선택
4. 시작일 입력
5. 휴가/교육/출장인 경우 종료일 입력 (선택)
6. `저장`

날짜를 클릭하면 해당 일자의 일정 목록이 팝업됩니다. 각 일정 옆 삭제 버튼으로 개별 삭제가 가능합니다.

## 6. 챗봇 사용법

### 6.1 챗봇 열기/닫기

- 화면 우측 하단의 로봇 아이콘(DK 로봇)을 클릭하면 챗봇 팝업이 열립니다.
- 열린 상태에서 `✕` 버튼 또는 팝업 상단 `✕`를 클릭하면 닫힙니다.
- 팝업이 열리면 입력란에 자동으로 포커스됩니다.

### 6.2 질문하기

- 입력란에 질문을 입력하고 Enter 키를 누르거나 전송 버튼을 클릭합니다.
- 추천 질문 칩을 클릭하면 해당 질문이 바로 전송됩니다.

### 6.3 추천 질문 (6개)

| 칩 | 설명 |
| --- | --- |
| 7월 계획과 실적은? | 특정 월의 계획/실적 상세 |
| 1분기 달성률 | 분기별 누적 달성률 |
| 상반기 구매 현황 | 1~6월 구매량/금액 합계 |
| 등급별 비중 | 등급별 물량 비중 비교 |
| 거래처별 실적 | 전체 거래처 납품 현황 |
| 전체 현황 요약 | 대시보드 전체 주요 지표 요약 |

추천 질문 칩은 한 번 클릭하면 사라집니다. `↺`(초기화) 버튼으로 대화를 초기화하면 다시 나타납니다.

## 8. 자주 묻는 상황

### 로그인되지 않을 때

- 아이디와 비밀번호를 다시 확인합니다.
- 계정 상태가 `비활성`인지 운영 담당자에게 확인합니다.
- 다른 브라우저나 시크릿 모드에서는 별도 로그인이 필요합니다.

### 데이터가 비어 보일 때

- 먼저 `조회 연도`가 맞는지 확인합니다.
- 해당 연도에 원본 실적 데이터가 아직 입력되지 않았을 수 있습니다.
- 계획 데이터가 없는 연도(예: 2023년)는 일부 차트/KPI가 표시되지 않을 수 있습니다.

### 보고서가 생성되지 않을 때

- 페이지를 새로고침한 뒤 다시 시도합니다.
- 브라우저 팝업/다운로드 차단 여부를 확인합니다.
- Chart.js CDN 접근이 가능한 네트워크인지 확인합니다.

### 차트가 보이지 않을 때

- 브라우저를 새로고침합니다.
- CDN(인터넷)에 접근 가능한 환경인지 확인합니다.
- 다른 탭으로 이동했다가 돌아오면 차트가 자동 리사이즈됩니다.

### 챗봇이 "질문을 이해하지 못했습니다"라고 할 때

- 보다 구체적인 키워드를 사용합니다 (예: 계획, 실적, 구매, 등급, 거래처, 수입, 배분).
- 기간을 명시합니다 (예: `7월`, `상반기`, `1분기`).
- 추천 질문 칩을 참고합니다.

## 9. 사용 유의사항

- 이 시스템의 저장 데이터는 현재 사용 중인 브라우저 환경(PC + 브라우저 + 프로필)에 저장됩니다.
- 다른 PC나 다른 브라우저에서는 동일한 데이터가 자동으로 공유되지 않습니다.
- 브라우저의 "사이트 데이터 삭제" 기능을 사용하면 저장된 운영 데이터가 삭제될 수 있습니다.
- 최신 Chrome 또는 Edge 브라우저를 권장합니다.
- 인쇄 시 헤더, 탭 메뉴, 푸터 등이 자동으로 숨겨집니다.
