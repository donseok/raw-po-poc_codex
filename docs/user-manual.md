# 사용자 매뉴얼

## 1. 목적
이 문서는 동국제강 원료기획팀 실적 모니터링 대시보드의 기본 사용법을 안내합니다. 화면 조회, 연도 전환, 보고서 내보내기, 공지와 일정 확인 같은 일상 사용 흐름을 기준으로 작성했습니다.

## 2. 접속과 로그인
1. 브라우저에서 `login.html`에 접속합니다.
2. 아이디와 비밀번호를 입력한 뒤 `로그인`을 누릅니다.
3. 기본 테스트 계정은 `dongkuk1 / 1234`입니다.
4. 빠르게 접속하려면 `테스트 계정 바로 입장` 버튼을 사용할 수 있습니다.
5. 사용자 상태가 `비활성`이면 로그인할 수 없습니다.

## 3. 메인 화면 구성
상단 헤더에서 다음 항목을 사용할 수 있습니다.

| 항목 | 설명 |
| --- | --- |
| 현재 날짜 | 접속 시점의 날짜와 요일을 표시합니다. |
| 조회 연도 | 연도를 바꾸면 차트, KPI, 표, 원본 입력 패널이 해당 연도로 갱신됩니다. |
| 사용자 표시 | 현재 로그인한 사용자 이름과 부서를 표시합니다. |
| 보고서 내보내기 | 현재 선택 연도 기준 DOCX 보고서를 생성합니다. |
| 로그아웃 | 현재 세션을 종료하고 로그인 화면으로 이동합니다. |

탭 메뉴는 다음과 같습니다.

| 탭 | 주요 용도 |
| --- | --- |
| 부재료실적 모니터링 | 월별 계획/실적, 거래처 추이, 핵심 포인트 확인 |
| 등급별현황/수입관리 | 구매량, 등급 구성, 수입 선적 현황 확인 |
| 거래처 관리 | 거래처 목록 조회 및 등록/수정/삭제 |
| 공지사항 | 공지 열람 및 등록, 팀 일정 확인 |
| 사용자관리 | 사용자 목록 조회 및 등록/수정/삭제 |

## 4. 공통 사용 방법

### 4.1 조회 연도 변경
1. 헤더의 `조회 연도` 목록을 엽니다.
2. 원하는 연도를 선택합니다.
3. 현재 활성 탭의 차트, KPI, 표가 자동으로 갱신됩니다.

### 4.2 표 정렬
정렬 가능한 표는 열 제목을 클릭하면 오름차순/내림차순으로 바뀝니다.

### 4.3 보고서 내보내기
1. `보고서 내보내기`를 클릭합니다.
2. 시스템이 차트를 캡처한 뒤 DOCX 파일을 생성합니다.
3. 보고서에는 `부재료실적 모니터링`, `등급별현황/수입관리` 내용이 포함됩니다.

### 4.4 챗봇 사용
우측 하단 챗봇 버튼을 누르면 대화창이 열립니다. 예시는 다음과 같습니다.

- `7월 계획과 실적은?`
- `상반기 구매량 추이는?`
- `현대스크랩 거래처 현황`
- `수입 선적 현황 보여줘`

## 5. 탭별 사용법

### 5.1 부재료실적 모니터링
이 화면에서는 월별 목표 대비 실적을 한 번에 확인할 수 있습니다.

- KPI 카드: 누계 목표, 누계 실적, 달성률 등 핵심 수치를 표시합니다.
- 월별 계획 대비 실적 차트: 월별 계획과 실적, 누계 달성 흐름을 보여줍니다.
- 월별 주요 거래처 납품 추이: 주요 거래처의 월별 추이를 확인합니다.
- 핵심 체크 포인트: 변동이 큰 구간이나 주목할 지표를 요약합니다.
- 월별 누계 실적 상세: 표 형식으로 월별 수치를 확인하고 정렬할 수 있습니다.

접힌 입력 패널 3종은 주로 운영 담당자가 사용합니다.

- `월별 계획 대비 실적 데이터`
- `거래처 납품 실적관리 데이터 (인천공장)`
- `검수 등급 매핑 관리`

일반 사용자는 조회 위주로 사용하고, 데이터 갱신이 필요하면 운영 담당자에게 요청하는 것을 권장합니다.

### 5.2 등급별현황/수입관리
이 화면에서는 구매량과 등급 구성, 수입 진행 현황을 확인합니다.

- 월별 구매량 추이 차트
- 인천공장 배분 현황
- 등급별 비교 표
- 월별 구매실적 상세 표
- 수입 계약/선적 현황 표

연도를 바꾸면 비교 기준과 표 내용도 함께 갱신됩니다.

### 5.3 거래처 관리
거래처 관리 탭에서는 등록된 공급업체 정보를 확인할 수 있습니다.

- KPI 영역에서 거래처 수, 월 공급능력, 연간 납품량, 평균 성과율을 확인합니다.
- `+ 거래처 등록`으로 신규 거래처를 추가할 수 있습니다.
- 목록의 수정/삭제 버튼으로 기존 거래처를 관리할 수 있습니다.

등록 항목은 코드, 거래처명, 지역, 대표자, 연락처, 신뢰등급, 월 공급능력, 금년 납품량입니다.

### 5.4 공지사항
공지사항 탭에서는 공지를 조회하고 등록할 수 있습니다.

- 목록에서 제목을 클릭하면 상세 내용을 확인할 수 있습니다.
- `필독` 체크를 하면 공지가 상단 고정됩니다.
- 공지 수정과 삭제에는 게시글 등록 시 입력한 비밀번호가 필요합니다.

### 5.5 팀 일정관리
공지사항 탭 오른쪽에는 팀 일정 캘린더가 있습니다.

- `+ 일정 등록` 버튼으로 팀원 일정을 추가합니다.
- 날짜를 클릭하면 해당 일자의 일정 목록을 볼 수 있습니다.
- 일정 유형은 오전반차, 오후반차, 반반차, 휴가, 교육, 출장, 외근 등을 지원합니다.
- 휴가, 교육, 출장은 시작일과 종료일을 함께 입력할 수 있습니다.

### 5.6 사용자관리
사용자관리 탭에서는 사용자 정보를 관리합니다.

- 전체 사용자, 활성 사용자, 관리자, 비활성 사용자 KPI를 확인합니다.
- `+ 사용자 등록` 버튼으로 신규 계정을 만들 수 있습니다.
- 기존 계정은 수정/삭제 버튼으로 관리합니다.
- 사용자 상태를 `비활성`으로 바꾸면 로그인할 수 없습니다.

## 6. 자주 묻는 상황

### 로그인되지 않을 때
- 아이디와 비밀번호를 다시 확인합니다.
- 계정 상태가 `비활성`인지 운영 담당자에게 확인합니다.

### 데이터가 비어 보일 때
- 먼저 `조회 연도`가 맞는지 확인합니다.
- 해당 연도에 원본 실적 데이터가 아직 입력되지 않았을 수 있습니다.

### 보고서가 생성되지 않을 때
- 페이지를 새로고침한 뒤 다시 시도합니다.
- 브라우저 팝업/다운로드 차단 여부를 확인합니다.

## 7. 사용 유의사항
- 이 시스템의 저장 데이터는 현재 사용 중인 브라우저 환경에 저장됩니다.
- 다른 PC나 다른 브라우저에서는 동일한 변경 내용이 자동으로 공유되지 않을 수 있습니다.
